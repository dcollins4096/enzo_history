The Enzo Source Code Flowchart and Code Browser

This web page links to the Enzo source code flowchart and a html code browser. The flowchart graphics and web pages (the pretty pictures and associated image maps) were created by David Collins using a program called Visual Thought. The html code browser utilizes a perl script called EnzoCodeFlow, which was written by Brian O'Shea. This script is freely downloadable and distributed under the GPL. If you find any inaccuracies in any part of this website, please email Brian O'Shea. As of the last update, this web page suite corresponds to the 0.8.0 version of Enzo (friendly user release).
